# 🌱 Sumati's Seeds of Wisdom

A modern, elegant, responsive personal blogging and wisdom-sharing website —
*where timeless wisdom meets everyday life.*

Built with **Next.js 14 (App Router)**, **TypeScript**, **Tailwind CSS**,
**Framer Motion**, and **Decap CMS**, deployable to **Netlify** in minutes.

---

## ✨ Features

- Soft pastel, minimalist, mobile-first design with rounded cards
- Sticky floating navbar with instant global search
- Hero, trending topics, sidebar widgets, and elegant article cards
- Markdown-based blog with reading time, ratings, tags, and related posts
- Article pages with table of contents, social sharing, prev/next, comments
- Categories, About (timeline + values + gallery), Contact (Netlify Forms)
- Newsletter signup via Netlify Forms with spam protection (honeypot)
- Decap CMS admin at `/sanjuoliveteddy` with Netlify Identity + editorial workflow
- Full SEO: dynamic metadata, Open Graph, Twitter cards, JSON-LD
  (`BlogPosting`, `BreadcrumbList`, `WebSite`), `sitemap.xml`, `robots.txt`
- Subtle Framer Motion animations (fade-in, hover, sticky navbar)

## ☁️ Deploy to Netlify

1. **Push to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Sumati's Seeds of Wisdom"
   git branch -M main
   git remote add origin https://github.com/<you>/sumatis-seeds-of-wisdom.git
   git push -u origin main
   ```

2. **Create the Netlify site**
   - New site → Import from GitHub → select the repo.
   - Build command: `npm run build`
   - Publish directory: `.next`
   - The `@netlify/plugin-nextjs` plugin (declared in `netlify.toml`) is
     installed automatically.

3. **Enable Netlify Identity (CMS login)**
   - Site → **Identity** → *Enable Identity*.
   - **Services → Git Gateway** → *Enable Git Gateway*.
   - Registration → set to *Invite only* (recommended).
   - Identity → **Invite users** → invite your email, accept the invite, set a
     password. Then log in at `/sanjuoliveteddy`.

4. **Enable Netlify Forms**
   - Forms are auto-detected from `public/__forms.html` on deploy.
   - View submissions under Site → **Forms** (`newsletter`, `contact`,
     `comments`).

5. **Update site URL & analytics**
   - Set `siteConfig.url` in `src/lib/site.ts` to your Netlify URL (or custom
     domain).
   - Update the Plausible `data-domain` in `src/app/layout.tsx`, or replace the
     Plausible script with your Google Analytics snippet.

## 🔍 SEO & Performance

- `sitemap.xml` and `robots.txt` are generated by `src/app/sitemap.ts` and
  `src/app/robots.ts`.
- Each page sets canonical URLs, Open Graph, and Twitter card metadata.
- Structured data (`BlogPosting`, `BreadcrumbList`, `WebSite`) is embedded as
  JSON-LD.
- Images use `next/image` (AVIF/WebP) for fast, optimized loading.
- Built to target **95+** Lighthouse scores across Performance, Accessibility,
  Best Practices, and SEO.
